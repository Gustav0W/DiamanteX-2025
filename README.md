# Gerador de Diamante	

![](https://i.imgur.com/Yvh9r42.gif)

## Introdução

É simplesmente um programa que gera um diamante, baseado no número que você escrever! É mais difícil do que parece fazer isso aqui.

## Funcionalidades

- **Validação de número ímpar**: Ele valida automaticamente se você colocou um número ímpar ou não;
- **Validação de número inteiro**: Ele valida automaticamente se você colocou um número inteiro;
- **Gera Diamantes**: Relaxa! Não são de verdade, mas ele gera "diamantes" na verdade são losangos de caracteres.

## Como Utilizar

1. Clone o repositório ou baixe o código fonte.
```
git clone https://github.com/Gustav0W/DiamanteX-2025
```
2. Abra o terminal ou o prompt de comando e navegue até a pasta raíz.
```
cd Diamante.ConsoleApp
```
3. Utilize o comando abaixo para restaurar as dependências do projeto.
```
dotnet restore
```
4. Em seguida compile com o comando: 
```
dotnet build --configuration Release
```

5. Execute com o comando:
```
dotnet run
```

6. Você verá essa mensagem:
```
Informe um número ímpar:
```
